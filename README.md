# 🧴 Terry-Nerrds Colognes | Fresh & Urban

Welcome to the official repository for **Terry-Nerrds Colognes**, an urban fragrance brand founded by **Terry Nerrd**. This site is designed to bring high-end, long-lasting scents to the bold and the stylish.

## ✨ Features
- **Dynamic Catalog**: Loads 24+ unique colognes from a JSON database.
- **Instant Search**: Real-time filtering by name, gender, or scent notes (Vanilla, Cedarwood, etc.).
- **User Authentication**: Integrated with **Clerk Auth** for a secure login experience.
- **Analytics**: Tracked via **Google Analytics (G-6LL5WXQTW8)**.
- **Contact System**: Fully functional inquiry form powered by **Formspree**.
- **Mobile First**: Built specifically to look great on all devices, including Android.

## 🛠️ Built With
- **Frontend**: HTML5, CSS3 (Cyberpunk/Urban Aesthetic), JavaScript (ES6)
- **Database**: JSON
- **Tools**: Developed on **Android (Honor X5b)** using **Termux**
- **Hosting**: GitHub Pages

## 📂 Project Structure
- `index.html`: The main shop floor.
- `about.html`: Our brand story and mission.
- `legal.html`: Ingredients and safety information.
- `style.css`: Custom "Neon-Dark" UI styling.
- `script.js`: Search logic and Clerk Auth integration.
- `catalog.json`: The source of truth for all product data.
